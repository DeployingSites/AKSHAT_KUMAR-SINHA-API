Fetching

To fetch the list of music you need to use fetch

Code

                        
                            
    await fetch("https://akshat-kumar-sinha-api.onrender.com/api/music-player/getSongs")
    .then((response)=> response.json()) 
    .catch((e)=>console.log(e))
